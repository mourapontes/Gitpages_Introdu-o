O que é o GitHub Pages?

Uma ferramenta gratuita para transformar seu repositório GitHub em um site acessível online!

Pré-requisitos

  • Conta no GitHub
  • Repositório com seu app web (HTML, CSS, JS)

Passo 1: Criar um repositório

Crie um novo repositório no GitHub com o nome do seu projeto.

Passo 2: Subir arquivos

Faça upload dos arquivos do seu site (index.html, etc) para o repositório.

Passo 3: Acessar configurações

Vá até o repositório, clique em Settings > Pages.

Passo 4: Ativar o GitHub Pages

Escolha a branch (geralmente main) e clique em Save.

Passo 5: Pronto!

Seu site será publicado em:

https://seunome.github.io/nomedorepo

Dicas úteis

  • Use index.html como página inicial
  • Evite usar caminhos absolutos ("/img")
  • Consulte a aba Actions se der erro
